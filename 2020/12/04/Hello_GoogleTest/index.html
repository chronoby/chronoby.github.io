Hello GoogleTest

简介

GoogleTest 是一个跨平台的 C++ 测试框架，支持各种平台上不同类型的测试。被 Chromium, LLVM 和 OpenCV 等项目所使用。

If you write C++ code, googletest can help you!

环境

  • OS: Ubuntu 20.10
  • CMake: 3.16.3
